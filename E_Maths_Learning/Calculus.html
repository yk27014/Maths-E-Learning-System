Stationary, Turning, Local Maximum and Local Minimum Points

A stationary point is a point on a curve where the value of is equal to 0. There are 3 different types of stationary point.

  • Turning Points: Is when the gradient changes from positive to negative or negative to positive.
  • Local Maximum Points: Have the highest value of y in a given interval.
  • Local Minimum Points: Have the lowest value of y in a given interval.
